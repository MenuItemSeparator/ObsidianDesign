Активация на ~

Основные команды:

В скобках [] обязательные аргументы
В скобках {} необязательные аргументы

| Команда    | Аргументы          | Описание                                                                                                          | Пример                 |
| ---------- | ------------------ | ----------------------------------------------------------------------------------------------------------------- | ---------------------- |
| spawn      | [pawn_id] [X] [Y]  | Создает пешку pawn_id на клетке X Y. Отсчет ведется из левого верхнего угла. [[Пешки]]                            | spawn Pawn_Citizen 2 3 |
| viewpoint  | [point_name]       | Перемещает камеру к заданному point_name. [[ViewPoints]]                                                          | viewpoint VP_TV        |
| kill       | [X] [Y]            | Убивает жителя (не пешку) на клетке X Y. Отсчет ведется из левого верхнего угла. Работает только в основной игре. | kill 2 3               |
| additem    | [item_id] {amount} | Добавляет в инвентарь игрока предмет item_id в количестве amount. [[Предметы]]                                    | additem Ladder 2       |
| teleport   | [X] [Y]            | Телепорт на клетку X Y. Отсчет ведется из левого верхнего угла.                                                   | teleport 2 1           |
| playermove | [N]                | Пешка игрока проходит N шагов.                                                                                    | playermove 3           |
| tileevent  | [event_id]         | Запускает событие клетки event_id. Чаще всего, просто переключает уровень в телевизоре. [[Tile Events]]           | tileevent CityGates    |
| setstat    | [stat] [value]     | Устанавливает stat героя на value. В игре только 2 стата: Health и MaxHealth                                      | setstat Health 0       |
Остальные команды не надо использовать :)
